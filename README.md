# AI Receipt Analyzer

Inteligentna aplikacja w Streamlicie służąca do automatycznej ekstrakcji, kategoryzacji i analizy wydatków z paragonów przy użyciu modelu **Gemini Pro (Vision)**.

## Funkcjonalności
* Skanowanie zdjęć paragonów i natychmiastowe wyciąganie pozycji oraz cen.
* SPrzypisywanie produktów do predefiniowanych kategorii budżetowych.
* Możliwość manualnej korekty cen, nazw oraz kategorii bezpośrednio w tabeli.
*  Opcja wykluczenia poszczególnych pozycji z ostatecznego podsumowania finansowego.
* **Analityka Live:** Dynamicznie odświeżany wykres kołowy wydatków.
