# 🧾 AI Receipt Analyzer

Inteligentna aplikacja w Streamlicie służąca do automatycznej ekstrakcji, kategoryzacji i analizy wydatków z paragonów przy użyciu modelu **Gemini Pro (Vision)**.

## 🚀 Funkcjonalności
* 📸 **Ekstrakcja Vision AI:** Skanowanie zdjęć paragonów i natychmiastowe wyciąganie pozycji oraz cen.
* 🏷️ **Automatyczny Triage:** Przypisywanie produktów do predefiniowanych kategorii budżetowych.
* 📝 **Interaktywny Edytor:** Możliwość manualnej korekty cen, nazw oraz kategorii bezpośrednio w tabeli.
* ❌ **Filtrowanie Wydatków:** Opcja wykluczenia poszczególnych pozycji z ostatecznego podsumowania finansowego.
* 📊 **Analityka Live:** Dynamicznie odświeżany wykres kołowy wydatków.
